Stock Market Simulator

This is a stock market simulation application built with Python and Flask. Key features include real-time stock data integration, portfolio tracking, and watchlist management.

Features:
  • Real-time stock price updates (30-minute intervals)
  • Portfolio tracking with P/L calculations
  • Buy/Sell functionality
  • Watchlist management
